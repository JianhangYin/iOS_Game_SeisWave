# SeisWave
This game is now available on apple store! Check more details on my personal website: www.jianhang.space

SeisWave is a not only a pixel art game but also an app aiming to teach geophysics student how to locate epicenter using the data recorded by the stations. In this game, you will start with deploying the stations. In general, three stations can confirm the location of earthquake. Then according to the distance between the source and each station calculated by the different arrival time of P-wave and S-wave, you will locate the epicenter during the game.
